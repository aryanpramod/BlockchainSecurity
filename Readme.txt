
It is based on a dataset for cyber security. Cloud services have increased the number of data owners
who keep their encrypted data there, while also increasing the number of users who access the data.
It uses a Flask module and is based on the block chain. using a hybrid of the ECC and AES algorithms 
with encrypted and decrypted data. Based on a keyword search algorithm, an encrypted file will be 
stored in a cloud server and accessible by the user. based on user Type the keyword together with 
the encrypted query. Searching an encrypted cloud server after that. Finally, retrieve the relevant 
files based on your query. Users who enter a specific key to decrypt files do so with superior 
results in terms of recollection, ranking privacy, precision, searching time, and other factors.
